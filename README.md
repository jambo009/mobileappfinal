# 🧊 스마트 냉장고 
> **"버려지는 식재료 제로! 자취생을 위한 스마트 유통기한 관리 솔루션"**

안드로이드 Jetpack Compose를 활용하여 자취생들이 냉장고 속 식재료의 유통기한을 직관적으로 관리하고, 식비를 절감할 수 있도록 돕는 실용적인 애플리케이션입니다.

## 🚀 개발 배경 및 의도 
많은 자취생이 냉장고 속 식재료의 유통기한을 놓쳐 음식을 버리는 경험을 합니다. 단순한 텍스트 메모의 한계를 넘어, **남은 일수를 자동으로 계산**하고 **시각적 경고**를 주는 전용 관리 도구가 필요하다고 판단하여 기획 및 개발했습니다.

---

## ✨ 핵심 기능 및 제작 단계 

### 1단계: 사용자 워크플로우 설계 
* 사용자가 앱을 켜자마자 '관리 시작하기' 버튼을 통해 집중 점검 세션을 시작할 수 있도록 설계했습니다.
* 상단 전원 버튼을 통해 점검을 종료하는 직관적인 흐름을 구축했습니다.

### 2단계: 유통기한 신호등 로직 구현 
* 등록된 식재료의 잔여 기한에 따라 배경 바의 색상이 자동으로 변하는 알고리즘을 적용했습니다.
    * 🔴 **빨강**: 만료/임박 (3일 이내)
    * 🟡 **노랑**: 주의 (7일 이내)
    * 🟢 **초록**: 안전 (8일 이상)

### 3단계: 실시간 데이터 요약 
* 상단 배너에 '유통기한 임박 식품 개수'를 자동으로 계산해 노출하여 사용자의 즉각적인 행동을 유도합니다.

## 🛠 사용 기술 및 기술적 이해 

### 1. 사용 기술
* **Language**: Kotlin
* **UI Framework**: Jetpack Compose (Material 3)
* **State Management**: `remember` 및 `mutableStateOf`를 이용한 반응형 UI

### 2. 기술적 도전 및 이해도 증명 (Troubleshooting)
* **상태 관리 원리 이해**: 리스트 아이템의 추가/삭제 시 화면이 즉시 갱신되도록 Compose의 **재구성(Recomposition)** 원리를 활용하여 리스트 상태를 업데이트했습니다.
* **API 레벨 호환성 해결**: `java.time` 라이브러리의 API 레벨 이슈를 해결하기 위해 시스템 타임스탬프(`System.currentTimeMillis`) 방식의 연산으로 코드를 최적화하여 하위 버전 기기 호환성을 확보했습니다.
* **가독성 있는 설계**: `FoodItem`(데이터), `Components`(UI), `MainActivity`(로직)로 파일을 분리하여 코드의 유지보수성을 고려한 설계를 진행했습니다.
